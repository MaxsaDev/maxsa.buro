## Основні можливості

### Меню за замовчуванням

Система підтримує автоматичне призначення меню користувачам:

- **Для нових користувачів**: Пункти меню з `is_default = true` автоматично призначаються при реєстрації
- **Для існуючих користувачів**: При встановленні `is_default = true` меню автоматично призначається всім існуючим користувачам через тригери БД
- **Автоматичне видалення**: При знятті `is_default = false` видаляються тільки автоматично призначені меню (`is_auto_assigned = true`), ручні призначення залишаються

Це дозволяє адміністратору легко налаштувати базове меню для всіх користувачів без необхідності індивідуального призначення.
